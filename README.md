# vue-express-forum
## 基于vuejs,express,mogodb的多人论坛webapp

### start app:

**install dependencies& devDependencies**

$ npm install

**start mongodb database server and listening at port 27017**

> net start mongodb

**start express server**

$DEBUG=vue-express-forum && npm start

**enjoy the app**

localhost:3000/

### development

$ npm run dev

### build

$ npm run build
